Predicción de Accidentes Cerebrovasculares
-----------------------------------------------------------------------------------------------------------
Descripción

Este proyecto se centra en la predicción de accidentes cerebrovasculares utilizando técnicas de aprendizaje automático. Se basa en un conjunto de datos de atención médica que incluye información sobre diversos factores de riesgo como la edad, el género, la presión arterial, el índice de masa corporal (IMC), entre otros.

-----------------------------------------------------------------------------------------------------------
Objetivo

El objetivo principal de este proyecto es construir un modelo predictivo que pueda identificar a las personas con alto riesgo de sufrir un accidente cerebrovascular. Esto permitirá una intervención temprana y la prevención de posibles complicaciones.

-----------------------------------------------------------------------------------------------------------
Metodología

1. **Recopilación de datos:** El conjunto de datos utilizado se obtuvo de Kaggle y se encuentra en formato CSV.
2. **Exploración de datos:** Se realizó un análisis exploratorio de datos (EDA) para comprender las características de los datos, identificar valores atípicos y visualizar las relaciones entre las variables.
3. **Preprocesamiento de datos:** Se imputaron los valores faltantes y se codificaron las variables categóricas. Además, se escalaron las variables numéricas utilizando StandardScaler.
4. **Selección de características:** Se seleccionaron las características más relevantes para el modelo predictivo.
5. **Modelado:** Se entrenaron diferentes modelos de aprendizaje automático como regresión logística, árboles de decisión y máquinas de vectores de soporte.
6. **Evaluación:** Se evaluó el rendimiento de los modelos utilizando métricas como precisión, recall y F1-score.
7. **Despliegue:** El modelo seleccionado se desplegará para su uso en un entorno real.

-----------------------------------------------------------------------------------------------------------
Resultados

Los resultados preliminares indican que el modelo de regresión logística tiene un buen rendimiento en la predicción de accidentes cerebrovasculares. Sin embargo, se están realizando más pruebas para optimizar el modelo y mejorar su precisión.
